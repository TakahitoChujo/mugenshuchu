# mugenshuchu
# 無限集中 
集中力を高め、作業効率を最大化する iOS & Apple Watch 対応ポモドーロタイマー

## 📌 概要  
「無限集中」は、iPhone と Apple Watch の両方で使えるポモドーロ・タイマーアプリです。  
- iPhone でデイリーの集中／休憩セッションをグラフで記録  
- Apple Watch でタイマーを手元でスタート／ストップ  
- iPhone と Watch 間でデータ同期（ログ連携）  
- シンプルで直感的な UI、集中（オレンジ）／休憩（ブルー）の色分け  

## 🎯 主な機能  
- 25 分集中 → 5 分休憩
- 今日の集中合計時間と休憩時間を可視化  
- 休憩時間カウントダウン機能  
- Apple Watch 単独でも操作可能（iPhone不要）  
- Watch／iPhone 間のログ自動同期  

## 🚀 動作環境  
- iOS 17 以降  
- watchOS 10 以降  
- Xcode 15 以降で開発・ビルド
